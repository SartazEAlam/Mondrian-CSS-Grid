# Mondrian-CSS-Grid
#  Mondrian Painting – CSS Grid

A recreation of a **Piet Mondrian–style abstract painting** built using **HTML and CSS Grid**.  
This project focuses on mastering **CSS Grid layout**, alignment, and spacing.


# Features

*  Pure HTML & CSS (no JavaScript)
*  CSS Grid–based layout
*  Mondrian-inspired color blocks
*  Precise grid alignment and spacing
*  Clean and readable code


# Technologies Used

~ HTML5
~ CSS3
~ CSS Grid
